## 🔧 技术实现

### **响应式设计：**
```vue
<el-col :span="24">  <!-- 全宽显示 -->
```

### **标签数据：**
```javascript
const activityTabs = [
  { label: '全部', value: 'all', icon: Menu },
  { label: '采购', value: 'purchase', icon: ShoppingCart },
  { label: '销售', value: 'sale', icon: Money },
  { label: '库存', value: 'inventory', icon: Box }
]
```

### **计算属性：**
```javascript
// 当前标签名称
const currentActivityTabLabel = computed(() => {
  const tab = activityTabs.find(t => t.value === currentActivityTab.value)
  return tab ? tab.label : ''
})

// 筛选后的动态
const filteredActivities = computed(() => {
  if (currentActivityTab.value === 'all') {
    return activities.value
  }
  return activities.value.filter(
    activity => activity.type === currentActivityTab.value
  )
})
```

### **过渡动画：**
