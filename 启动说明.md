# 项目启动说明

## 后端启动步骤

1. 确保已安装JDK 1.8+和Maven 3.6+
2. 确保MySQL和Redis服务已启动
3. 修改 `supermarket-backend/src/main/resources/application.yml` 中的数据库配置
4. 在项目根目录执行：
   ```bash
   cd supermarket-backend
   mvn clean install
   mvn spring-boot:run
   ```
5. 后端服务将在 http://localhost:8080 启动
6. API文档地址：http://localhost:8080/api/doc.html

## 前端启动步骤

1. 确保已安装Node.js 16+
2. 在项目根目录执行：
   ```bash
   cd supermarket-frontend
   npm install
   npm run dev
   ```
3. 前端服务将在 http://localhost:3000 启动

## 注意事项

- 数据库和Redis需要先启动
- 第一阶段已完成基础框架搭建，后续需要完成数据库设计和业务功能开发
