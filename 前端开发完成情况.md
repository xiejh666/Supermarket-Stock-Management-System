## 🚀 启动指南

### 开发环境启动

#### 方法一：使用批处理脚本（推荐）
```bash
# 双击运行
supermarket-frontend/启动开发服务器.bat
```

#### 方法二：手动启动
```bash
cd supermarket-frontend
npm install
npm run dev
```

访问：http://localhost:3000

---

### 生产环境构建

#### 方法一：使用批处理脚本
```bash
# 双击运行
supermarket-frontend/构建生产版本.bat
```

#### 方法二：手动构建
```bash
cd supermarket-frontend
npm run build
```

构建产物位于 `dist/` 目录
